Set Up Cloud Sync
One-time setup (5 mins). After this, you and Emil will always see the same data!
1
On your phone, open Safari and go to:
script.google.com
Sign in with your Google account.
2
Tap "New Project" (top left). Delete all the existing code in the editor.
3
Tap the code block below to copy it, then paste it into the editor: function doGet(e){ var action=e.parameter.action||'read'; var key=e.parameter.k||'data'; var props=PropertiesService.getScriptProperties(); if(action==='save'){ var val=e.parameter.v||''; props.setProperty(key,val); return ContentService.createTextOutput('ok') .setMimeType(ContentService.MimeType.TEXT); } var data=props.getProperty(key); return ContentService.createTextOutput(data||'null') .setMimeType(ContentService.MimeType.TEXT); } function doPost(e){ var props=PropertiesService.getScriptProperties(); try{ var payload=JSON.parse(e.postData.contents); props.setProperty(payload.k, payload.v); }catch(err){ props.setProperty('tracker_state', e.postData.contents); } return ContentService.createTextOutput('ok') .setMimeType(ContentService.MimeType.TEXT); } 👆 Tap to copy · then paste into Apps Script
4
Tap the 💾 Save icon (floppy disk). Name the project "JE Tracker".
5
Tap Deploy → New Deployment.
• Type: select Web App
• Execute as: Me
• Who has access: Anyone
Tap Deploy. Authorize when asked.
6
Copy the Web App URL it shows (starts with https://script.google.com/macros/...) and paste it below!
